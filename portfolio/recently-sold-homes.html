Recently Sold Homes

The Washington Post


This application managed the details of homes that had been recently sold.

Role: Senior Web Developer

Team: Senior Front End Web Developer, Front End Web Developer, Lead Back End Developer, Senior Back End Developer

Jan 2012 - July 2014


Activities Performed: Code Architecture and Development, Review High Fidelity Prototypes


Technology Stack: jQuery, Java, Oracle SQL Database


Background

The legacy system was out of date and heavily dependent on Perl scripting; it was no longer maintainable. The manager wanted to rebuild the application. After I attended An Event Apart 2012, I convinced the team to build one of the first responsive websites at The Washington Post.

Design

I collaborated closely with the UX Designer to design an application that was accessible on mobile devices, tablets, and larger viewports.

Development

I was the lead on this project and determined the architecture of the front end code. I worked closely with the Java developer to refine the APIs that retrieved and posted the data. At the time, I wrote the code entirely in jQuery. Some of the feature highlights include the following:

  • Complex user form that contained elements that needed to be draggable on a touch screen device
  • Markers on Google Maps - if there was a large group of houses in an area it would cluster them as a group with a count
  • Used browser local and session storage to maintain search form options and filtered results
